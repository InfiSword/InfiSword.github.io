2. 중앙 집중식 입력 시스템 (InputManager)

수백 개의 유닛이 개별 이벤트를 수신하지 않고 InputManager가 모든 입력을 가로채 상황에 맞는 객체에 전달합니다.

// InputManager.cs: 레이캐스트를 통한 유닛 검출 일원화 private IInteractable GetInteractableUnderMouse() { RaycastHit2D unitHit = Physics2D.Raycast(m_mouseWorldPosition, Vector2.zero, 0f, unitLayerMask); if (unitHit.collider != null) { return unitHit.collider.GetComponent<IInteractable>(); } return null; }

3. 그리드 기반 논리 시스템 (FileGridManager)

월드 좌표를 그리드 인덱스로 실시간 매핑하여 유닛의 배치 가능 여부와 스냅 기능을 구현했습니다.

// FileGridManager.cs: 그리드 유닛 배치 시도 public bool TrySetUnitAtGrid(File_Base unit, FileGrid targetGrid) { if (targetGrid.obstacleObject != null) return false; unit.transform.position = targetGrid.transform.position; unit.transform.SetParent(targetGrid.transform); targetGrid.SetFileUnit(unit); return true; }
